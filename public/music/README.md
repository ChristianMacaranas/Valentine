# Music Files

Place your Valentine's Day music files in this directory.

## Recommended Music Files

1. **valentine-song.mp3** - Main romantic background music
2. **celebration.mp3** - Victory celebration sound (optional)

## How to Add Music

1. Place your MP3 file in this `/public/music/` directory
2. Update the `src` prop in `App.tsx`:
   ```tsx
   <MusicPlayer src="/music/your-song-name.mp3" />
   ```

## Music Sources

- **Free Royalty-Free**: 
  - Pixabay Music
  - YouTube Audio Library
  - Freesound.org
- **Premium**: 
  - Epidemic Sound
  - Artlist.io
  - Musicbed

## File Format

- **Format**: MP3 recommended (best browser compatibility)
- **Size**: Keep under 5MB for faster loading
- **Quality**: 192kbps is good balance
